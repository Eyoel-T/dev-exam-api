# dev-exam-api

api for dev-exam-frontend

## Available Scripts

In the project directory, you can run:

### `pnpm i`

install project dependency

### `pnpm dev`

Runs the app in the development mode.\
Open [http://localhost:5000](http://localhost:5000) to view it in the browser.
